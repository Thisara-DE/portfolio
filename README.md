## 🚀 Quick Start

### Installation

```bash
# Clone the repository
git clone https://github.com/Thisara-DE/portfolio.git
cd portfolio

# Install dependencies
npm install
```

### Usage

```bash
# Start development server
npm start

# Run tests
npm test

# Build for production
npm run build
```
